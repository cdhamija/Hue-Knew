# Hue Knew? 🎨

**Smaragdine?! Coquelicot?! WHO KNEW?!**

▶ **[Play it now → cdhamija.github.io/Hue-Knew](https://cdhamija.github.io/Hue-Knew/)**

A tiny tap toy about the colors nobody told you about. Tap anywhere and a
brand-new color springs onto the screen with a satisfying *boop*, a little
buzz, and its gloriously obscure name in big bold Helvetica. Tap it again and
it dances — bounces, wiggles, pops, shimmies, or does a full 3D flip, because
colors contain multitudes.

There are **64 rare colors** to discover. Zaffre. Gamboge. Drake's-Neck
(a real color, named after an actual duck's actual neck). Eigengrau — the
color you see when you close your eyes. Caput Mortuum, which translates to
"dead head" and is somehow a lovely brown. Collect them all and earn the
coveted title of *hue genius*.

## ✨ The good stuff

- 🎵 **Every color sings its own note** — hue is mapped onto a pentatonic
  scale, so mashing the screen accidentally composes music. You cannot play
  a wrong note. Mozart could never.
- 💥 **Five dance moves**, dealt at random. The 3D flip is objectively the
  best one.
- 📳 **Real haptics on iPhone** — new colors thump, re-taps tick.
- 🏆 **A collection screen** that remembers every find forever, with
  mysterious `?` slots taunting you about the ones you haven't met yet.
- 🧊 **Zero dependencies in the app itself.** One HTML file. No framework.
  The entire game ships in less space than this repo's `node_modules` uses
  for its license files.

## 🚀 Run it

```
npm install
npm run serve        # → http://localhost:4173 — go tap things
```
